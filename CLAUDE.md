# Claude Instructions for SpicebushWebapp

## Memory Management

**IMPORTANT**: Always use the `journal/` directory to maintain session memory and context across conversations. For every significant interaction, decision, or piece of information discovered:

1. Create or update markdown files in `journal/` directory
2. Use descriptive filenames with dates (e.g., `journal/2025-07-15-project-setup.md`)
3. Log key findings, decisions made, problems solved, and context discovered
4. Reference previous journal entries when relevant to maintain continuity
5. Update existing journal entries rather than creating duplicates when adding related information

This ensures continuity and context preservation across all sessions.
