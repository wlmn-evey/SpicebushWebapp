# Spicebush Documentation

This directory contains the active documentation set for the Netlify-first Spicebush webapp.

## Start Here
- `docs/DOCUMENTATION_INDEX.md`
- `docs/refactor-master-plan.md`
- `docs/roadmap.md`

## Core Runtime Stack
- Frontend + server rendering: Astro
- Backend platform: Netlify
- Database: Neon via Netlify DB connection string
- Admin auth: magic-link flow with DB-backed sessions

## Purpose of This Folder
- Keep only active operational and engineering docs in `docs/`.
- Remove outdated/one-off plans that conflict with current scope.
- Store incident notes under `docs/incidents/`.
