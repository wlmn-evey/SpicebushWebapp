## FIT (Family Individualized Tuition) Model

The live website confirms Spicebush uses a **Family Individualized Tuition (FIT)** model with the following key features:

1. **Income-Based Tiers**: Four tuition levels (A-D) based on family size and gross household income
2. **Reserved Slots**: Specific number of spaces reserved for each income tier
3. **Sliding Scale**: Tuition ranges from $18,035 down to $5,397 for full-day programs
4. **Subsidy Support**: Tuition D families are encouraged to apply for state child care subsidies
5. **Extended Care Variations**: Extended care rates also vary by tuition tier, with Tuition D families receiving free extended care
