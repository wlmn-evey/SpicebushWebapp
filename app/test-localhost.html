What to Test:

  • Images Loading: All images should display properly (WebP with JPG fallbacks)
  • Responsive Behavior: Resize browser window to see focal points adjust
  • Logo Display: Check header and footer for logo
  • Performance: Images should load quickly with appropriate sizes

Troubleshooting:

  • If pages don't load, ensure server is running: npm run dev
  • Check browser console for any JavaScript errors
  • Use Developer Tools Network tab to verify image loading
  • Test in different browsers for compatibility
