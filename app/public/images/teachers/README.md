## Instructions

1. Right-click each source URL above and "Save Image As..."
2. Save with the specified filename in this directory
3. Optimize images for web (recommended: 800px width, 80% JPEG quality)
4. After adding photos, run the database migration to update the URLs

## Image Specifications

- **Format:** JPEG (.jpg)
- **Recommended Size:** 800px width (height will auto-scale)
- **Quality:** 80% compression for web optimization
- **Naming:** Use kebab-case matching teacher slugs
